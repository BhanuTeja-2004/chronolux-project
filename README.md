# ChronoLux — Full Stack Watch Store

## Features
- Premium UI (HTML, CSS, JS)
- Spring Boot REST APIs
- MySQL database
- User Registration & Login

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Java, Spring Boot
- Database: MySQL

## How to Run

### Backend
1. Open project in IntelliJ
2. Configure MySQL:
   - DB name: chronolux
   - Update username/password in `application.properties`
3. Run `ChronoluxApplication.java`

### Frontend
1. Open `frontend/index.html` with Live Server
2. Use Register → Login

## Notes
- API runs on: http://localhost:8081
